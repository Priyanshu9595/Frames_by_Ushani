# Frames by Ushani

A luxury photography and filmmaking portfolio showcasing cinematic weddings, corporate events, reels, and emotional visual stories. 

## Features
- **Responsive Design**: Mobile-friendly layout ensuring a beautiful viewing experience across all devices.
- **Cinematic Lightbox**: An immersive media viewer for both photos and videos.
- **Dynamic Header & Footer**: Context-aware navigation and a fully responsive footer.
- **Contact Integration**: Seamless inquiry form for bookings via Formspree.

## Tech Stack
- React 19
- Tailwind CSS
- Framer Motion
- Lucide React
- Vite

## Setup
To run this project locally:
1. Clone the repository
2. Run `npm install` to install dependencies
3. Run `npm run dev` to start the development server
